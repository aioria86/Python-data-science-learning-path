# Python Data Science Learning Path

This repository contains projects and exercises from the **Maven Analytics Python for Data Science Learning Path**.

The goal of this repository is to practice real-world data science workflows using Python, including data preparation, exploratory data analysis, machine learning, and natural language processing.

---

## Learning Path Structure

1. **Data Prep & EDA**
2. **Regression**
3. **Classification**
4. **Unsupervised Learning**
5. **Natural Language Processing**

---

## Projects Included

- Bank Customer Data Preparation
- Predicting Fuel Economy
- Bank Customer Classification
- Bank Customer Segmentation

---

## Setup

Create a virtual environment:

python -m venv venv


Activate it:


venv\Scripts\activate


Install dependencies:


pip install -r requirements.txt
